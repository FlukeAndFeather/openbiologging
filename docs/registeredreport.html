Trends, challenges, and opportunities in open biologging data

Introduction (504/500 words)

Biologging is the practice of using animal-borne sensors to remotely observe animals and their environments1. In the six decades since a biologist first attached a modified kitchen timer to a seal2, biologging has become an invaluable, multidisciplinary tool used by ecologists3, climate scientists4, oceanographers5, and more. As biologging data continue to grow in size (e.g., over six billion records in Movebank6) and complexity (e.g., novel sensors like infrasound recorders7), “big data” methods have been highlighted as a key direction for future developments8,9. Simultaneously, biologging data have been proposed as valuable conservation tools for specific taxa (e.g., seabirds10 and sharks11) and entire regions (e.g., the Southern Ocean12). If large, synthesis biologging datasets are going to have such important impacts on ecological research and ecosystem conservation, then equitable access to these data is a matter of practical and ethical concern.

Open science practices, such as publishing data and code alongside manuscripts, increase transparency and improve efficiency13,14. The “openness” of data is determined by multiple factors. The FAIR principles15 - findability, accessibility, interoperability, and reusability - provide a framework for facilitating open data sharing. FAIR data are a critical component of groundbreaking synthesis science16,17 and effective conservation practices18,19 NOTE. Moreover, for fields like ecology and geosciences where a lack of diversity reflects disproportionately hostile treatment of excluded groups21,21,22, FAIR data can serve an important role in transformations towards inclusive research cultures23,24 (a transformation the open science movement itself is currently undertaking25,26).

There have been numerous efforts within the biologging research community to promote open data sharing practices. Essential cyberinfrastructure include domain-specific data repositories (Movebank6, Seabird Tracking Database seabirdtracking.org, Euromammal27) and data standards28–30, which facilitate adoption of the FAIR principles (Table 1).

Table 1: The FAIR principles, as implemented in biologging cyberinfrastructure, support data sharing and improve research transparency and efficiency.
FAIR principle Definition Application to biologging Example
Findability Data and metadata have a globally unique and persistent identifier (e.g., a digital object identifier, DOI) and are indexed in a searchable resource Data repositories, like the Movebank Data Repository, improve data discoverability and may assign DOIs to data (avoiding issues with broken hyperlinks, for example) A tracking dataset31 deposited in the Movebank Data Repository is findable by searching in the repository or by its permanent DOI, 10.5441/001/1.293
Accessibility Data and metadata are retrievable by open and universal protocols, such as HTTP Data repositories allow scientists to retrieve biologging data via a web browser or other open source tools Publicly available data on Movebank may be downloaded via the website or with the move R package32
Interoperability Data use formal and shared formats and vocabularies Data standards like the Movebank data model and data formats like nc-eTAG reduce barriers to combining datasets and increase uptake within and across scientific disciplines Standardizing location and environmental data from seal-borne biologgers facilitated their reuse by oceanographers to study polar regions5
Reusability Data and metadata are richly defined and reuse permissions are clearly defined Data repositories and standards together capture essential context for biologging data and provide licensing options for data reuse The Movebank data model records whether an instrumented animal was manipulated (e.g., relocated before release), which would affect the results of many ecological analyses

Despite these biologging cyberinfrastructure developments, the general state of open data practices within the community remains unknown33 NOTE. We will conduct a systematic review of the biologging literature to establish a baseline of open data practices. The review will quantify the context of biologging studies (e.g., instrumented species and sensor types) and their data sharing practices. By data sharing practices, we refer to how biologging studies both share data (e.g., were the data assigned a persistent identifier such as a DOI?) and utilize other sources of open data (e.g., satellite remote sensing data).
